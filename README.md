✨ **Live Demo:** [Portfolio Link](https://github.com/himmatmundhe07/Portfolio_New)

---

## 🌟 Key Features

-   **Dynamic 3D Aesthetics:** Integrated Spline models and floating 3D elements for a premium feel.
-   **Immersive Animations:** Powered by `framer-motion` for smooth layout transitions and scroll-driven effects.
-   **Deep Personalization:** Features a dynamic "About Me" section with personalized greetings and interactive mascot.
-   **Comprehensive Project Showcase:** Categorized projects including Functional Apps, Website Clones, Games, and Hackathon entries.
-   **Fully Responsive:** Optimized for all devices—from desktop monitors to mobile screens.
-   **Glassmorphism Design:** Modern UI with subtle gradients, blurs, and border glows.

---

## 🛠️ Tech Stack

-   **Frontend:** React 19, JavaScript (ES6+), HTML5, CSS3 (Vanilla + Tailwind)
-   **Animations:** Framer Motion, Spline (3D)
-   **Icons:** Lucide React
-   **Styling:** Tailwind CSS (Custom Design System)
-   **Tools:** Vite, Git, Postman

## 🚀 Getting Started

To run this project locally, follow these steps:

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/himmatmundhe07/Portfolio_New.git
    cd Himmat
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Run Development Server:**
    ```bash
    npm run dev
    ```

4.  **Build for Production:**
    ```bash
    npm run build
    ```
